# Cherry Production - Music Portfolio Frontend

Cherry Production's personal music portfolio website. Here, you can listen to and search my transcription or arrangements of Kpop songs, and download the corresponding scores.

## Live Demo

**Website:** [https://cherryproduction.pages.dev](https://cherryproduction.pages.dev)

## Features

- **Search & Filter** - Find songs by title, artist, or types, quickly find your target piece
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Internationalization (i18n)** - Multi-language support (English, Traditional Chinese, Simplified Chinese)
- **Mobile-Friendly** - Touch-friendly interface for mobile users
- **Dynamic Content** - Fetch songs, audio, online video links and score from REST API
- **SEO Optimized** - robots.txt and sitemap.xml included

## Tech Stack

| Technology | Purpose |
|------------|---------|
| **React 18** | Frontend framework |
| **React Router DOM** | Client-side routing |
| **React i18next** | Internationalization |
| **Axios** | HTTP client for API calls |
| **Bootstrap 5** | UI components and grid system |
| **Bootstrap Icons** | Icon library |
